## Technical Description
- **`scripts/pdp_content.py`** (pur, fără DB/random): din `attributes` (usage/key_benefit/key_ingredients/not_recommended_for/fragrance_free/spf) + `reviewSummary` → `sections()`, `ingredient_list()`, `badges()`, `reviews()`. Chei stabile (`slug`, `external_id`) → idempotent.
- **`product_sections`** (kind usage/benefits/ingredients/warnings): din usage + key_benefit + key_ingredients + not_recommended_for. Poziționate.
- **`ingredients` + `product_ingredients`**: INCI normalizat din `key_ingredients` (upsert pe `(business_id, slug)`, `is_key=true`, `position`).
- **`product_badges`**: derivate din atribute reale (`fragrance_free`→„Fără parfum", `spf`→„Cu SPF N", `rating≥4.7`→„Best-seller").
- **`reviews`** individuale: 2-3/produs din `reviewSummary` (pros→pozitiv, con→mixt), `source='seed_demo'`, autor demo determinist, `external_id` stabil.
- **Seed:** extinde `seed_catalog_v2.py::_upsert_product` — idempotent (delete + reinsert pe chei stabile), în TX, scoped `DEMO_BIZ` (`admin_conn`).
