# NX-174 — P1: comparație pur referențială, deterministă peste `displayed_products`

## Context
Audit live 2026-07-17 — confirmat pe cod + reprodus de `scripts/sim/nx172_smoke.py` (FINDING
non-fatal, [nx172_smoke.py:198-206](../scripts/sim/nx172_smoke.py#L198)).

Flux: „arată-mi două fonduri de ten" → „compară primele două".
1. Triajul (nano) **moștenește** categoria (`fond-de-ten`) și o re-atașează ca filtru pe turul de
   comparație.
2. Gate-ul deterministic cere `not route.filters`
   ([deterministic.py:211-218](../src/agent/deterministic.py#L211)) → cu filtre moștenite, calea
   deterministă (`agent_compared`, tabel structurat IZI-parity) **nu se declanșează**.
3. Se cade pe LLM: modelul trebuie să copieze UUID-urile din context → a copiat greșit UUID-ul
   celui de-al doilea produs de două ori → `compare_products` a primit 2 id-uri identice →
   `get_products_by_ids` a întors 1 produs → `error="need_2"`
   ([catalog_tools.py:925](../src/tools/catalog_tools.py#L925)).
4. Reply-ul a vorbit despre două produse, dar a afișat unul.

Cauza de fond: o cerere **pur referențială** („primele două" — se rezolvă integral din
`displayed_products`) e lăsată pe seama copierii de UUID-uri de către LLM.

## Definition of Done
- [ ] O comparație pur referențială rulează determinist peste `displayed_products`, **chiar dacă**
      triajul reaplică filtre moștenite (gate-ul `not route.filters` nu mai o dezactivează).
- [ ] Nu depinde de copierea UUID-urilor de către LLM.
- [ ] Rehidratează produsele fresh din DB (`business_id` scoped) și **păstrează ordinea** cerută.
- [ ] Dacă sunt două produse valide → reply-ul/tabelul conțin **ambele**.
- [ ] Distinge „referențial" (moștenire OK) de „constrângere NOUĂ" (ex. „compară-le pe cele sub 100
      lei" = filtru real → nu forța calea deterministă).
- [ ] Test E2E cu **triaj realist** (filtre moștenite pe turul 2) + categorie moștenită.
- [ ] Test: `compare_products` cu id-uri duplicate → nu mai produce „vorbește de 2, afișează 1"
      (degradare explicită, nu `need_2` tăcut).
- [ ] Gate verde.
