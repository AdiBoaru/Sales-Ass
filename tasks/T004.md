# T004 — CI: ruff (lint + format)
**Owner:** S · **Faza:** MVP · **Zi:** Z1 · **Branch:** `chore/T004-ci-ruff` · **Complexitate:** XS · **Estimare:** 1h

## Goal
Primul status check obligatoriu: stil și erori statice prinse înainte de review. Elimină toată categoria de discuții „pune spațiul aici" din PR-uri.

## Business Context
Cod consistent = review rapid = viteza promisă în planul de 2 săptămâni.

## Technical Description
`.github/workflows/ci.yml` cu job `lint`: checkout → setup-python 3.12 → cache pip → `pip install ruff` → `ruff check .` → `ruff format --check .`. Configurarea ruff în `pyproject.toml`: `line-length = 100`, `target-version = "py312"`, select implicit + `I` (isort). Jobul `test` se adaugă în T005 în același workflow.

## Principii CLAUDE.md aplicabile
Suport pentru regula 5 din CONTRIBUTING (teste/verificări obligatorii pe PR).

## Implementation Steps
1. Scrie `pyproject.toml` cu secțiunea `[tool.ruff]`.
2. Scrie workflow-ul cu jobul `lint`.
3. PR — verifică-l că rulează chiar pe acest PR.
4. După merge: adaugă `lint` la required checks (T002 pasul 2).
