# NX-40 — pg_partman: partiții automate pe messages + analytics_events
**Owner:** S · **Faza:** MVP · **Zi:** Z3 · **Branch:** feat/NX-40-partman · **Complexitate:** XS · **Estimare:** 3h

## Goal
Partiția lunii viitoare există întotdeauna, creată automat, cu alertă dacă nu — niciodată un INSERT care crapă pe 1 ale lunii la 00:01.

## Business Context
`messages` și `analytics_events` sunt partiționate lunar, dar partițiile se creează manual în migrări. Asta e o bombă cu ceas operațională: prima lună în care nimeni nu rulează migrarea = toate mesajele tuturor clienților pică simultan.

## Technical Description
Supabase are `pg_partman` + `pg_cron` ca extensii gata de activat. `partman.create_parent()` peste tabelele NATIVE deja partiționate (`p_type => 'native'`, `p_control => 'created_at'`, `p_interval => 'monthly'`, `p_premake => 2`) — preia managementul partițiilor existente. `pg_cron` programează `partman.run_maintenance_proc()` zilnic la 03:00. Retenția NU se setează aici (ștergerea e treaba politicii GDPR/T177, decizie separată) — `p_retention` rămâne NULL. Alerta: jobul aplicației (scheduler) verifică zilnic că partiția pentru luna următoare există în `pg_catalog`; dacă nu → eveniment + Slack.

## Principii CLAUDE.md aplicabile
P6 (infrastructura nu pică tăcut), P9 (operare repetabilă, nu ritualuri manuale).

## Implementation Steps
1. Migrare `0xx_partman.sql`: `CREATE EXTENSION IF NOT EXISTS pg_partman; CREATE EXTENSION IF NOT EXISTS pg_cron;` + `SELECT partman.create_parent('public.messages', 'created_at', 'native', 'monthly', p_premake => 2);` idem `analytics_events`.
2. `SELECT cron.schedule('partman-maint', '0 3 * * *', $$CALL partman.run_maintenance_proc()$$);`
3. Rulează manual o dată `run_maintenance_proc()` → verifică `\d+ messages` că partițiile +1 și +2 luni există.
4. `src/jobs/check_partitions.py`: query pe `pg_inherits`/`pg_class` pentru partiția lunii următoare pe ambele tabele; lipsă → `analytics_events('partition_missing')` + alertă.
5. Notă în `docs/runbook.md`: ce faci dacă alarma sună (comanda de creare manuală).
