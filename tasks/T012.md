# T012 — CLAUDE.md + CONTRIBUTING.md în repo
**Owner:** J · **Faza:** MVP · **Zi:** Z1 · **Branch:** `docs/T012-claude-md` · **Complexitate:** XS · **Estimare:** 0.5h

## Goal
Memoria permanentă a proiectului intră în repo: arhitectura (CLAUDE.md) și regulile de lucru (CONTRIBUTING.md). Claude Code le citește automat la fiecare sesiune.

## Business Context
Fiecare sesiune de Claude Code fără context complet = cod care încalcă arhitectura = timp pierdut la review.

## Technical Description
Copiezi versiunile finale (CLAUDE.md actualizat cu [002] + CONTRIBUTING.md) în root. Adaugi folderul `tasks/` cu cardurile primite (batch 1 acum, restul pe parcurs). Orice schimbare ulterioară de arhitectură = PR pe CLAUDE.md cu review obligatoriu (e acoperit de CODEOWNERS dacă adaugi `CLAUDE.md @senior` în T003 — recomandat, fă-o aici).

## Principii CLAUDE.md aplicabile
Toate — ăsta e fișierul care le definește.

## Implementation Steps
1. Copiază fișierele în root + folderul tasks/.
2. Adaugă linia `CLAUDE.md @senior-user` în CODEOWNERS.
3. PR + merge.
