# T038 — Script backup/restore local
**Owner:** J · **Faza:** P1 (O1) · **Branch:** `chore/T038-backup-script` · **Complexitate:** XS · **Estimare:** 2h

## Goal
Independență de backup-ul Supabase: un script care trage dump complet local și unul care îl restaurează într-un Postgres de test. Folosit și ca să-ți faci mediu local identic cu staging-ul când vrei să spargi ceva fără frică.

## Business Context
PITR-ul Supabase e plasa principală (T018/T162); dump-ul local e plasa plasei + unealtă de dev.

## Technical Description
`scripts/db_backup.sh`: `pg_dump "$DIRECT_URL" -Fc -f backups/nativx_$(date +%F).dump` (custom format, comprimat). `scripts/db_restore_local.sh`: pornește un container postgres:16 efemer + `pg_restore` în el + afișează connection string-ul local. Backups/ în .gitignore. Atenție la versiunea pg_dump local vs server (16) — documentat în script.

## Implementation Steps
1. Cele două scripturi + gitignore pe backups/.
2. Rulare completă o dată: dump + restore + SELECT count pe products în containerul local.
3. PR.
