# T013 — Meta developer app + WhatsApp test number
**Owner:** S · **Faza:** MVP · **Zi:** Z1 (PRIMUL task al zilei!) · **Branch:** — · **Complexitate:** S · **Estimare:** 1.5h

## Goal
Deblochează tot ce ține de WhatsApp: app Meta + numărul de test din sandbox, care e disponibil INSTANT (spre deosebire de numărul de producție). Fără asta, M1 (echo e2e, ziua 5) nu există.

## Business Context
Tot produsul vostru trăiește pe acest canal. Sandbox-ul vă duce până la demo; producția (T016/T251) vine separat.

## Technical Description
developers.facebook.com → Create App → tip Business → adaugă produsul WhatsApp. Primești automat: un **test phone number** cu **Phone Number ID** și un **temporary access token** (24h — pentru azi ajunge; token permanent prin System User în Business Settings → de făcut tot azi: System User cu rol admin, generate token cu `whatsapp_business_messaging`, `whatsapp_business_management`, fără expirare). Adaugă numerele voastre personale ca **recipient phone numbers** (sandbox-ul trimite doar către max 5 numere verificate). Notează în vault-ul vostru de secrete: APP_SECRET (Settings → Basic), ACCESS_TOKEN, PHONE_NUMBER_ID.

## Principii CLAUDE.md aplicabile
P7 în spirit: token-ul e per-canal → va sta în `core.channel_instances`/secrete, nu hardcodat.

## Implementation Steps
1. Creează app-ul Business + produs WhatsApp.
2. System User + token permanent (NU rămâne pe cel de 24h!).
3. Adaugă ambele telefoane ca destinatari de test.
4. Trimite hello world din Graph API Explorer / curl către telefonul tău.
5. Salvează cele 3 secrete în .env local (NU în repo).
