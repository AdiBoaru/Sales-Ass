**Owner:** Claude (build) / Codex (verify) · **Faza:** P1 · **Track:** Response Quality · **Branch:** `feat/NX-182-relaxed-disclosure` · **Complexitate:** S/M · **Estimare:** 1-2 zile

## Goal
Mini-Match-Gate cu ROI mare, FĂRĂ epicul QuerySpec: retrieval-ul expune **care hard field a fost relaxat** de `_relax_ladder`, iar compose adaugă un **disclosure determinist**. Nu prezenta niciodată un rezultat relaxat drept „potrivire exactă". Se livrează înainte de/în paralel cu V2.

## Business Context
Azi `_relax_ladder` (`src/tools/catalog_tools.py:163`) relaxează cumulativ concerns→category→features, dar singurul semnal spre răspuns e `relevance`/off-category la nivel de **categorie** — insuficient. Un „fără parfum, sub 80" supra-constrâns poate întoarce un produs cu parfum, prezentat ca și cum s-ar potrivi.

## Contract
**Un singur nume peste tot: `relaxed_constraints: list[RelaxedConstraint]`** (NU `relaxed_fields`, NU `list[str]` — insuficient pentru disclosure exact).

```python
@dataclass(frozen=True)
class RelaxedConstraint:
    facet_key: str        # cheia canonică a fațetei relaxate
    relaxed_step: int     # treapta din ladder la care s-a relaxat
    original_value: str | None  # NORMALIZAT (niciodată text liber brut / PII); None dacă nenormalizabil
```

- **Display labels:** NX-182 include un **registru MINIMAL de etichete de afișare** per fațetă (RO/EN/HU, doar labels — fără tipuri/operatori). NX-186 îl absoarbe și îl extinde la registrul tipizat complet. Fațetă fără label → fallback pe `facet_key`.
- **PII/free-text:** `original_value` se stochează DOAR normalizat (vocabular DomainPack); text liber nenormalizabil → None (se afișează doar numele fațetei).
- Price/stock rămân dure sub sort_mode; brand niciodată relaxat → nu apar în listă.

## Technical Description
- `_relax_ladder` / caller: urmărește ce trepte au fost aplicate → `relaxed_constraints`.
- Propagă în `RetrievalResult` + în `RichReply`/proză.
- Disclosure determinist per-locale în compose (label din registrul minimal); **suprimă eticheta de exact**; cardurile rămân ca alternative apropiate.
- Generic pe vertical.
